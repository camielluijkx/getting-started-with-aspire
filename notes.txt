.NET Aspire Tutorial - Build Cloud-Ready Apps with .NET 10 and PostgreSQL

1. Update EF tools

	Update global EF CLI tool

		dotnet tool update --global dotnet-ef

	Verify installed version

		dotnet ef --version

2. Create new migration

	Add a new (initial or seed) migration

		dotnet ef migrations add SeedInitialBooks

	Implement the Up and Down methods

3. Run the application

	Apply database migration

		Database.Migrate() is automatically called at application startup

4. pgAdmin is available at http://localhost:5050

	Login:

		Email Address / Username:		admin@admin.com
		Password:						dbadminsecret

	Servers:

		Register > Server...

			General
				Name: PostgresLocal

			Connection
				Host name/address: postgres
				Port: 5432
				Maintenance database: booksDb
				Username: dbuser
				Password: dbusersecret

		PostgresLocal > Databases > booksDb > Query Tool

			SELECT * FROM public."Books"
			ORDER BY "Id" ASC
